# pcap-test
Packet Capture Application

## usage
```shell
ugonfor@gongon:~/ugonfor/pcap-test$ make
g++    -c -o pcap-test.o pcap-test.cpp
g++   pcap-test.o main.o  -lpcap -o pcap-test
ugonfor@gongon:~/ugonfor/pcap-test$ sudo ./pcap-test eth0
```
